# do-list

Create the basic structure for the todo app:

- Add a header with the title "My Todo List"
- Add a main content area 
- In the main content area, add a subheading "Add a new todo"
- Add a footer with the text "Created with GPT Engineer"

Use a clean and friendly design with a blue and white color scheme.
